# GraduateSchoolEntranceExamination

## 概要

東京大学大学院情報理工学系研究科の入試問題の過去問、および、共通数学の過去問の解答をまとめたものです。

[HP](https://www.i.u-tokyo.ac.jp/edu/course/mi/admission.shtml)も参照して下さい。
